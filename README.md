Este proyecto consiste en el desarrollo de un sistema web para la gestión de planes turísticos, orientado a facilitar la administración de clientes, planes, ventas y usuarios dentro de una empresa de turismo. El sistema fue desarrollado aplicando una arquitectura cliente-servidor, permitiendo una separación clara entre la lógica de presentación y la lógica de negocio.

La aplicación permite el registro y gestión de clientes con información personal y de contacto, así como la administración de planes turísticos, incluyendo tarifas, puntos de visita y características del viaje. Cada cliente puede realizar una compra que puede incluir uno o varios planes, quedando registrada la información detallada de la venta, como fechas de viaje, alimentación, número de personas y costo total.

El sistema incorpora un control de acceso por roles, diferenciando entre Súper Administrador y Vendedor. El Súper Administrador tiene permisos para gestionar planes, tarifas, puntos de visita y usuarios del sistema, mientras que el Vendedor puede administrar clientes y registrar ventas. Este enfoque permite un manejo seguro y organizado de la información.

Para la persistencia de los datos, el sistema utiliza una base de datos relacional, en la cual se almacenan de forma estructurada los registros de clientes, planes, usuarios y compras. Las operaciones CRUD (Crear, Leer, Actualizar y Eliminar) garantizan la correcta gestión de la información, asegurando integridad y consistencia en los datos.

Adicionalmente, el sistema permite la generación de reportes y estadísticas básicas, que facilitan el seguimiento de ventas y la toma de decisiones. La aplicación fue desarrollada como un proyecto académico, aplicando buenas prácticas de programación y diseño de software.

Funcionalidades principales:

Gestión de clientes (registro, edición y eliminación)

Administración de planes turísticos

Registro de ventas con múltiples planes por compra

Control de usuarios y roles

Persistencia de datos en base de datos relacional

Operaciones CRUD

Reportes y estadísticas básicas

Objetivo del proyecto:

El objetivo principal del proyecto es desarrollar una solución web que permita organizar y automatizar la gestión de una empresa de turismo, aplicando conceptos de bases de datos, seguridad básica, arquitectura cliente-servidor y desarrollo de aplicaciones web.
